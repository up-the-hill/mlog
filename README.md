# mlog

A simple cli tool to log atomic thoughts.

## Features

- A 128 character limit.
- Timestamped logs.
- Markdown export.

## Configuration file

Config file stored in OS-default config directory.

```config.toml
musings_file = "$HOME/mlog/musings.ndjson"
export_path = "$HOME/mlog/musings.ndjson"
char_limit = 128

```
